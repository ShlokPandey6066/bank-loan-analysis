# Bank Loan Analysis Dashboard

## Project Overview

This project analyzes loan application data using SQL, Python, Excel, and Power BI.

## Tools Used

- Excel
- SQL (MySQL)
- Python
  - Pandas
  - NumPy
  - Matplotlib
- Power BI

## Workflow

1. Data Cleaning using Python
2. Exploratory Data Analysis (EDA)
3. SQL Querying and Analysis
4. Dashboard Development in Power BI

## Key Insights

- Total Applications: 2000
- Approved Loans: 342
- Employed applicants received the highest number of approvals.
- Credit score influences loan approval outcomes.

## Files

- `loan_prediction_dataset.csv` - Original dataset
- `loan_cleaned.csv` - Cleaned dataset
- `bank_loan_analysis.sql` - SQL queries
- `loan_analysis.py` - Python analysis
- `Bank_Loan_Analysis.pbix` - Power BI dashboard
